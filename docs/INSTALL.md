# Vanessa-Notch 安装说明(发给朋友的版本)

在刘海处实时显示**网易云音乐**当前歌曲的逐字歌词。

## 环境要求
- **Apple Silicon Mac(M 系列)** —— 当前 DMG 仅含 arm64,Intel Mac 暂不支持
- macOS 13 (Ventura) 及以上;刘海机型最佳(非刘海屏会在屏幕顶部居中显示)
- 需安装并使用**网易云音乐**

## 安装
1. 双击 `Vanessa-Notch.dmg` 打开
2. 把 **Vanessa-Notch** 图标拖到 **Applications**(应用程序)文件夹
3. 从「应用程序」启动 Vanessa-Notch

## 首次打开被拦截?(正常现象)
本 App 未经 Apple 公证,首次打开会提示「无法打开,因为无法验证开发者」。任选一种方式放行(只需一次):

- **方式 A:右键打开**
  在「应用程序」里**右键点 Vanessa-Notch →「打开」→** 再点「打开」。

- **方式 B:系统设置**
  打开后若被拦,去 **系统设置 → 隐私与安全性**,在底部点 **「仍要打开」**。

- **方式 C:终端一行命令**(最省事)
  ```bash
  xattr -dr com.apple.quarantine /Applications/Vanessa-Notch.app
  ```
  之后即可正常双击打开。

## 使用
- 启动后**没有 Dock 图标**,只在**菜单栏右侧**出现一个圆角均衡条图标。
- 用网易云**播放歌曲** → 刘海下方出现黑色岛形面板,显示封面 + 音频条 + 逐字歌词。
- **菜单栏图标**:左键打开设置;右键弹出「偏好设置 / 退出」。
- **刘海面板**:右键同样可弹出「偏好设置 / 退出」。
- 设置里可调:歌词字号、面板宽度/高度、开机启动等。

## 读不到歌词 / 显示警告?
- macOS 15.4+ 首次可能需要授权读取「正在播放」信息;若菜单栏/胶囊显示**警告态**,按提示在 **系统设置 → 隐私与安全性** 放行后重启 App。
- 冷门歌曲匹配不到歌词时,会降级显示「歌名 - 歌手」;纯音乐显示「♪ 纯音乐」。
- 需要联网获取歌词;断网时只显示「歌名 - 歌手」,联网后切歌重试。

> 说明:音频跳动条是装饰动效,不代表真实频谱。本工具使用网易云非公开接口与系统私有接口,仅供个人/开源使用,无法上架 App Store。
